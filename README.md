# 소개팅 서비스 기획안

프리미엄 검증 기반 소개팅 서비스와 AI 정밀 분석 매칭 아이디어를 정리하는 저장소입니다.

핵심 방향은 단순 프로필 매칭이 아니라, 사람의 성향과 맥락을 AI가 분석하고 "왜 이 사람이 잘 맞는지"를 설명해주는 큐레이션형 매칭 서비스입니다.
