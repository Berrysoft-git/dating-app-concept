# 회원 등급 정책서

상태: 제품/운영 초안
기준 문서: `docs/service-overview.md`, `docs/prd.md`, `docs/mvp-scope.md`, `docs/privacy-verification-data-policy.md`
대상 서비스: 프리미엄 검증 기반 AI 소개팅 서비스

## 1. 목적

이 문서는 서비스의 회원 등급, 계정 상태, 권한 차이, 추천 노출 원칙을 정의한다.
핵심은 “누가 더 우월한가”를 정하는 것이 아니라, 검증 수준과 노출 동의 범위에 따라 안전하고 설명 가능한 추천을 운영하는 것이다.

## 2. 기본 원칙

- 회원 등급은 서비스 접근 권한과 추천 후보군을 제어하기 위한 운영 단위다.
- 셀렉트/프리미엄 등급은 단순 프로필 인증이 아니라 강한 자격 검증을 통과한 상태다.
- 일반 회원은 셀렉트/프리미엄 회원 목록을 직접 조회할 수 없다.
- 일반 회원은 명시적으로 opt-in한 경우에만 셀렉트/프리미엄 회원에게 추천 후보로 들어갈 수 있다.
- 인기 일반 회원은 “셀렉트 회원”이 아니며, 활동 품질과 신뢰 지표가 좋은 일반 회원을 의미한다.
- 등급명과 설명 문구는 소득, 직업, 학력, 외모의 우열 표현으로 쓰지 않는다.
- 등급 변경, 검증 승인/반려, 제한, 강등은 감사 로그를 남긴다.

## 3. 회원 등급과 계정 상태 분리

회원 등급과 계정 상태는 분리한다.

| 구분 | 의미 | 예시 |
|---|---|---|
| 회원 등급 | 추천 풀, 노출 권한, 검증 수준 | 일반, 인기 일반, 셀렉트/프리미엄 |
| 계정 상태 | 서비스 이용 가능 여부 | 활성, 심사 대기, 제한, 탈퇴 예정, 탈퇴 완료 |
| 검증 상태 | 셀렉트/프리미엄 신청 진행 상태 | 미신청, 신청 대기, 승인, 반려, 만료 |
| 노출 동의 | 특정 추천 풀에 포함될지 여부 | 셀렉트 추천 opt-in, opt-out |

예를 들어 일반 회원이 셀렉트 추천 opt-in을 했더라도 등급은 일반 회원이다. 반대로 셀렉트/프리미엄 검증이 승인되어도 계정이 제한 상태라면 추천 후보군에 포함되지 않는다.

## 4. 회원 등급 정의

### 일반 회원

정의:

- 기본 가입과 필수 프로필 작성을 완료한 회원이다.
- 성향/가치관 설문을 완료하면 일반 추천 후보군에 포함될 수 있다.

가능 기능:

- 기본 프로필 작성/수정
- 사진 등록
- 성향/가치관 설문
- 일반 추천 카드 수신
- 추천 카드 수락/거절/숨김
- 신고/차단
- 셀렉트 추천 opt-in 신청 및 철회
- 셀렉트/프리미엄 검증 신청

제한:

- 셀렉트/프리미엄 회원 목록을 직접 조회할 수 없다.
- opt-in하지 않으면 셀렉트/프리미엄 회원에게 추천 후보로 들어가지 않는다.
- 검증 뱃지나 셀렉트 전용 노출 권한을 갖지 않는다.

### 인기 일반 회원

정의:

- 일반 회원 중 프로필 완성도, 응답 품질, 신고 이력, 피드백, 운영자 평가가 좋은 회원이다.
- 셀렉트/프리미엄 회원과 같은 강한 자격 검증을 통과했다는 의미는 아니다.

산정 후보 지표:

- 프로필 완성도
- 사진 정책 통과 여부
- 설문 완료도
- 추천 카드 응답률
- 신고/차단 비율
- 운영자 품질 피드백
- 부정행위 이력

가능 기능:

- 일반 회원 기능 전체
- 추천 후보군에서 품질 신호로 반영
- opt-in한 경우 셀렉트/프리미엄 회원에게 더 우선적으로 추천될 수 있음

제한:

- 셀렉트/프리미엄 회원 목록을 직접 조회할 수 없다.
- opt-in하지 않으면 셀렉트/프리미엄 추천 후보군에 포함되지 않는다.
- 인기 일반 상태는 공개 배지로 과도하게 노출하지 않는다.

### 셀렉트/프리미엄 회원

정의:

- 강한 자격 검증 기준을 통과한 회원이다.
- 검증 유형은 직업, 소득/자산, 전문 자격, 학력/경력, 공개 영향력 등으로 분리한다.
- 검증 상태는 원본 인증자료가 아니라 심사 결과, 검증 유형, 유효기간, 감사 로그 중심으로 관리한다.

가능 기능:

- 셀렉트/프리미엄 검증 뱃지
- 셀렉트/프리미엄 추천 풀 참여
- 셀렉트/프리미엄 회원 또는 opt-in 일반/인기 일반 회원 추천 수신
- 더 세밀한 추천 노출 범위 설정
- 추천 카드 수락/거절/숨김
- 신고/차단

제한:

- 다른 셀렉트/프리미엄 회원 전체 목록을 무제한 조회하는 기능은 MVP에 포함하지 않는다.
- 검증 뱃지는 우열 표현이 아니라 검증 상태 표시로만 사용한다.
- 검증 자격이 만료되거나 재검증에 실패하면 셀렉트/프리미엄 추천 풀에서 제외된다.

## 5. 계정 상태 정의

| 상태 | 의미 | 추천 후보 포함 |
|---|---|---|
| 활성 | 정상 이용 가능 | 가능 |
| 프로필 미완성 | 필수 프로필 또는 설문 미완료 | 불가 |
| 심사 대기 | 셀렉트/프리미엄 검증 심사 중 | 일반 추천은 가능, 셀렉트 권한은 불가 |
| 재제출 요청 | 검증 자료 보완 필요 | 일반 추천은 가능, 셀렉트 권한은 불가 |
| 제한 | 신고, 정책 위반, 운영 판단으로 제한 | 불가 |
| 탈퇴 예정 | 계정 삭제 요청 후 보존 기간 중 | 불가 |
| 탈퇴 완료 | 계정 삭제 완료 | 불가 |

## 6. 추천 노출 정책

### 일반 회원 추천

- 일반 회원은 기본적으로 일반 추천 풀에 포함될 수 있다.
- 프로필 미완성, 제한, 탈퇴 예정, 신고 처리 중인 계정은 추천 후보에서 제외한다.
- 인기 일반 여부는 후보군 우선순위 신호로만 사용한다.

### 셀렉트/프리미엄 회원 추천

- 셀렉트/프리미엄 회원은 셀렉트/프리미엄 추천 풀에 포함될 수 있다.
- 셀렉트/프리미엄 회원의 공개 범위 설정을 우선 적용한다.
- 셀렉트/프리미엄 회원 전체 목록을 일반 회원에게 공개하지 않는다.

### 일반 회원의 셀렉트 추천 opt-in

- 일반 회원은 opt-in한 경우에만 셀렉트/프리미엄 회원에게 추천 후보로 포함될 수 있다.
- opt-in 화면에는 노출 범위, 철회 가능성, 셀렉트 회원 목록 비공개 원칙을 고지한다.
- opt-in 철회 후에는 신규 셀렉트/프리미엄 추천 후보군에 포함하지 않는다.
- 이미 생성된 추천 카드의 처리 방식은 별도 노출 정책서에서 확정한다.

## 7. 등급 변경 흐름

| 흐름 | 조건 | 결과 |
|---|---|---|
| 신규 가입 -> 일반 회원 | 필수 프로필과 설문 완료 | 일반 추천 후보 가능 |
| 일반 회원 -> 인기 일반 후보 | 활동 품질 지표 충족 | 추천 우선순위 신호 반영 |
| 일반 회원 -> 셀렉트 심사 대기 | 검증 신청 완료 | 심사 큐 진입 |
| 셀렉트 심사 대기 -> 셀렉트/프리미엄 | 심사 승인 | 셀렉트 권한 부여 |
| 셀렉트 심사 대기 -> 재제출 요청 | 자료 부족 또는 불명확 | 보완 요청 |
| 셀렉트 심사 대기 -> 반려 | 기준 미충족 | 셀렉트 권한 미부여 |
| 셀렉트/프리미엄 -> 일반 회원 | 검증 만료, 재검증 실패, 사용자 요청 | 셀렉트 권한 회수 |
| 모든 등급 -> 제한 | 정책 위반, 신고 처리, 운영 판단 | 추천 후보 제외 |

## 8. 관리자 처리 원칙

- 등급 변경은 사유 코드와 처리자를 남긴다.
- 셀렉트/프리미엄 승인, 반려, 재제출 요청은 감사 로그를 남긴다.
- 인기 일반 상태는 자동 지표만으로 확정하지 않고, 운영자 검토 또는 안전 필터를 거친다.
- 운영자는 특정 회원을 임의로 셀렉트/프리미엄으로 승격할 수 없고, 검증 절차를 통과해야 한다.
- 제한 상태 해제는 원 처리 사유와 재검토 기록을 확인한 뒤 수행한다.

## 9. AI 추천 입력 기준

AI 추천 feature에는 다음을 사용할 수 있다.

- 익명 후보 ID
- 회원 등급 범주
- 검증 상태 또는 검증 뱃지 여부
- 성향/가치관 요약 feature
- 추천 노출 동의 상태
- 추천 카드 상호작용 이력의 비식별 요약

AI 추천 feature에는 다음을 넣지 않는다.

- 인증자료 원본
- 고유식별번호 원문
- 소득/자산 증빙 원문
- 관리자 내부 메모 원문
- 민감한 사생활 추정 정보
- 등급을 우열로 해석하는 문구

## 10. MVP에서 확정할 항목과 이후 문서

MVP에서 확정:

- 일반, 인기 일반, 셀렉트/프리미엄의 권한 차이
- 일반 회원의 셀렉트 추천 opt-in/철회
- 셀렉트/프리미엄 회원 목록 비공개 원칙
- 계정 제한 상태의 추천 후보 제외
- 등급 변경 감사 로그

이후 문서에서 확정:

- 셀렉트/프리미엄 검증 기준표
- 인기 일반 회원 선별 기준의 세부 점수
- 노출 정책서
- 매칭 정책서
- API/DB 설계 초안
- 관리자 심사/운영 정책서
